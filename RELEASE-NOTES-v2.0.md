# 📥 วิธีติดตั้ง

### ลงใหม่ (ยังไม่เคยลงม็อดนี้)

1. โหลด **`LegendOfMortal-Thai-by-Krirk0137-v2.0.zip`** จากหัวข้อ **Assets** ด้านล่าง
2. แตกไฟล์ (extract) จะได้ `BepInEx/`, `winhttp.dll`, `doorstop_config.ini`, `Kanit_sdf` ฯลฯ
3. เปิดโฟลเดอร์เกม
   - ปกติอยู่ที่ `C:\Program Files (x86)\Steam\steamapps\common\Legend_of_Mortal\`
   - หาไม่เจอ → คลิกขวาเกมใน Steam → **Properties → Installed Files → Browse**
4. **คัดลอกทุกอย่างที่แตกออกมา ไปวางในโฟลเดอร์เกม**
   วางถูกแล้วจะเห็น `Legend_of_Mortal\BepInEx\`, `Legend_of_Mortal\winhttp.dll` อยู่ระดับเดียวกับ `Mortal.exe`
5. เปิดเกม → หน้า Title เลือก **ออปชั่นที่ 2**
6. คลิกดรอปดาวน์ด้านบน → เลือก **ออปชั่นที่ 2 (จีนตัวย่อ / Simplified Chinese)**
   ระบบจะสวมทับด้วยภาษาไทยให้เอง

### อัปเกรดจาก v1.x

แนะนำให้ **ลบโฟลเดอร์ `BepInEx` เดิมทิ้งก่อน** แล้วค่อยวางชุดใหม่ (ไฟล์ค้างเก่าอาจทำให้แปลเพี้ยน)
**เซฟเกมอยู่คนละที่ ไม่ได้รับผลกระทบ**

ของใหม่ที่เพิ่มมาคือโฟลเดอร์เดียว: `BepInEx\plugins\LomThaiText\`

- `LomThaiText.dll` — ปลั๊กอิน
- `Thai.tsv` — ตารางคำแปล 71,523 บรรทัด (เปิดแก้ด้วย Notepad ได้)

> ระบบแปลเดิมยังทำงานอยู่ครบ **ไม่ได้ถูกแทนที่** ทั้งสองทางทำงานร่วมกัน

# ↩️ ถ้าไม่ชอบ / อยากย้อนกลับ

**ลบโฟลเดอร์ `BepInEx\plugins\LomThaiText` ทิ้ง** → กลับไปเป็นพฤติกรรมแบบ v1.6 ทันที

ปลั๊กอิน **ไม่แก้ไฟล์ของตัวเกมแม้แต่ไบต์เดียว** ทุกอย่างเกิดขึ้นในหน่วยความจำตอนเล่นเท่านั้น
ถ้าจะถอนม็อดทั้งหมด ก็ลบ `BepInEx\`, `winhttp.dll`, `doorstop_config.ini` ออกจากโฟลเดอร์เกม

# ⌨️ ปุ่มลัดในเกม (สำหรับคนอยากปรับเอง)

| ปุ่ม | ทำอะไร |
|---|---|
| **Ctrl + F10** | โหลดคำแปลและค่าขนาดตัวอักษรใหม่ **ทันที ไม่ต้องปิดเกม** |
| **Ctrl + F11** | บันทึกตารางข้อความทั้งหมดของเกมออกมาเป็นไฟล์ |

### อยากแก้คำแปลเอง

เปิด `BepInEx\plugins\LomThaiText\Thai.tsv` ด้วย Notepad — แต่ละบรรทัดคือ
`รหัส` + **แท็บ** + `คำแปลไทย` แก้ฝั่งขวาแล้วเซฟ → กลับเข้าเกมกด **Ctrl+F10** เห็นผลเลย

> ⚠️ บันทึกเป็น **UTF-8** และห้ามแก้ฝั่งซ้าย (รหัส) เด็ดขาด

### ตัวหนังสือเล็กไป / ล้นกล่อง

กล่องข้อความในเกมออกแบบมาสำหรับอักษรจีน 2–4 ตัว ภาษาไทยยาวกว่าจึงถูกย่อลง
แก้ได้ที่ `BepInEx\Translation\th\Text\UI.resizer.txt` รูปแบบ:

```
/เส้นทางของกล่องข้อความ=AutoResize(true,ขนาดต่ำสุด,ขนาดสูงสุด)
```

**ไม่ต้องเดาเส้นทางเอง** — ระหว่างเล่น ปลั๊กอินจะเขียนรายการกล่องที่ยังไม่ได้ตั้งค่าไว้ให้ที่
`BepInEx\LomThaiText_unruled.tsv` พร้อมเส้นทางและขนาดกล่องจริง
แก้เสร็จกด **Ctrl+F10** เห็นผลทันที
