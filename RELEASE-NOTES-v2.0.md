# v2.0 — เปลี่ยนวิธีแปลทั้งระบบ: จาก "จับคู่ข้อความ" เป็น "ใส่ตรงตามรหัสของเกม"

**ดาวน์โหลด:** `LegendOfMortal-Thai-by-Krirk0137-v2.0.zip`
เกมเวอร์ชัน `1.0.5000.13` (2025-10-10) · แปลจีน → ไทย ครบทั้งเกม

---

## สรุปสั้น ๆ

ทุกเวอร์ชันที่ผ่านมา (v1.0–v1.6) ทำงานแบบ **"เห็นข้อความจีนแล้วเอาไทยไปทับ"**
รอบนี้เพิ่ม **ปลั๊กอินตัวใหม่** ที่เขียนคำแปลไทย **ลงในตารางข้อความของเกมโดยตรง โดยอ้างอิงรหัสประจำบรรทัด**

ฟังดูเป็นเรื่องเทคนิค แต่มันปลดล็อกสิ่งที่วิธีเดิม**ทำไม่ได้เลยไม่ว่าจะพยายามแค่ไหน** — และทำให้ม็อดนี้
**รอดจากการอัปเดตเกมในอนาคต**

> **คำแปลไม่ได้ถูกทำใหม่** — 71,412 จาก 71,523 บรรทัดคือคำแปลชุดเดิมจาก v1.6 ทั้งหมด
> เปลี่ยนแค่ "ท่อ" ที่ส่งคำแปลเข้าเกม แปลเพิ่มด้วยมือแค่ **111 บรรทัด** ที่วิธีเดิมเก็บไม่ได้จริง ๆ

**ครอบคลุม 71,523 จาก 72,883 รหัสข้อความของเกม = 98.13%**

## ⭐ สามอย่างที่วิธีเดิมทำไม่ได้ตลอดกาล

### 1. ข้อความซ้ำที่ต้องแปลต่างกัน — 12,761 รหัส (17% ของทั้งเกม)

คำว่า `好。` ในเกมมีอยู่ **214 รหัสแยกกัน** พูดโดยคนละตัวละคร คนละสถานการณ์
แต่วิธีเดิมจับคู่ด้วย**ข้อความ** ทั้ง 214 จุดจึงถูกบังคับให้แปลเหมือนกันชั่วนิรันดร์

ตอนนี้แต่ละรหัสมีคำแปลของตัวเอง **แยกกันได้แล้ว**

### 2. ข้อความที่เป็นอักษรจีนตัวเดียว

`唐` แปลว่า "ถัง" แต่ถ้าใส่ไว้ ระบบเดิมจะไปแทนที่ในคำอื่นด้วย — `三师兄` กลายเป็น `สาม师兄`
เพราะแบบนี้ v1.2 จึงต้อง **ปิดการแปลอักษรเดี่ยวทิ้งไป 97 บรรทัด**

วิธีใหม่ไม่มีการค้นหาข้อความเลย จึงปลอดภัยสนิท — รอบนี้เปิดกลับมาครบ:

| ก่อน | หลัง |
|---|---|
| ปี 三 十月中旬 | **ปีที่ 3 กลางเดือนตุลาคม** |
| 修 / 练 (ปุ่มแนวตั้ง) | **ฝึก / ฝน** |
| 锻 / 造 | **ตี / เหล็ก** |
| 食 · 衣 · 住 · 行 (ตัวเลือก) | **อาหาร · เสื้อผ้า · ที่อยู่ · เดินทาง** |
| 唐门暗器总纲 (6 ช่องซ้อน) | **สำนัก·ถัง·อาวุธ·ลับ·ตำรา·ใหญ่** |

### 3. ข้อความที่มีตัวเลขเปลี่ยนได้

`近战伤害{0:N0}  爆击率{1:P0}` — ตัวเลขเปลี่ยนทุกครั้ง จึงไม่มีทางจับคู่ได้
`NEXT_STEPS` เคยบันทึกไว้ว่า "รูปแบบไม่จำกัด แก้ไม่ได้"

ตอนนี้ **บรรทัดเดียวครอบคลุมทุกค่า** → `ความเสียหายประชิด 10  อัตราคริติคอล 0%`

## 🛡️ รอดจากการอัปเดตเกม

**วิธีเดิมพังเงียบ ๆ**: ถ้าผู้พัฒนาแก้ประโยคจีนแม้แต่ตัวเดียว บรรทัดนั้นจะหยุดจับคู่ทันที
แล้วตกไปให้ Google แปลมั่ว — โดยไม่มีอะไรเตือน คุณจะรู้ก็ตอนเล่นไปเจอเอง

**วิธีใหม่**: รหัสประจำบรรทัดไม่เปลี่ยนตามเนื้อหา อัปเดตเกมแล้วคำแปลเดิมยังทำงานต่อ
และมีเครื่องมือบอกได้แน่นอนว่าอะไรเปลี่ยน แยกเป็น 3 กอง:

| กอง | ความหมาย |
|---|---|
| **ADDED** | รหัสใหม่ที่ไม่เคยมี → แปลเพิ่ม |
| **CHANGED** | รหัสเดิมแต่**ต้นฉบับจีนถูกเขียนใหม่** → คำแปลไทยเก่าไปแล้ว ต้องแปลซ้ำ |
| **REMOVED** | รหัสถูกลบ |

กอง **CHANGED** คือสิ่งที่วิธีเดิมมองไม่เห็นเลย

## 📦 วิธีติดตั้ง / อัปเกรด

เหมือนเดิมทุกอย่าง — **คัดลอกทับโฟลเดอร์เกม** (ไม่ต้องลบของเก่า)

ของใหม่ที่เพิ่มมาคือโฟลเดอร์เดียว: `BepInEx/plugins/LomThaiText/`

- `LomThaiText.dll` — ปลั๊กอิน
- `Thai.tsv` — ตารางคำแปล 71,523 บรรทัด (แก้ไขได้ด้วย Notepad)

**ระบบเดิมยังทำงานอยู่ครบ ไม่ได้ถูกแทนที่** — ทั้งสองทางทำงานร่วมกัน
ถ้าไม่ชอบก็แค่**ลบโฟลเดอร์ `LomThaiText` ทิ้ง** แล้วกลับไปเป็น v1.6 ทันที
ปลั๊กอิน**ไม่แก้ไฟล์เกมแม้แต่ไบต์เดียว**

### ปุ่มลัดในเกม (ถ้าอยากปรับเอง)

| ปุ่ม | ทำอะไร |
|---|---|
| **Ctrl + F10** | โหลดคำแปลและค่าขนาดตัวอักษรใหม่ **ทันที ไม่ต้องปิดเกม** |
| **Ctrl + F11** | บันทึกตารางข้อความทั้งหมดของเกมออกมาเป็นไฟล์ |

แก้คำแปลใน `Thai.tsv` แล้วกด Ctrl+F10 เห็นผลเลย — เหมาะกับการช่วยกันเก็บคำที่ยังไม่สวย
