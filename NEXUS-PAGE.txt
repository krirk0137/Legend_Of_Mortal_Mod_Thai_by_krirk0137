[line]

[size=5][b]นี่คืออะไร[/b][/size]

ม็อดแปลภาษาไทย (ไม่เป็นทางการ) สำหรับเกม [b]Legend of Mortal (活侠传)[/b]
แปลจาก [b]ภาษาจีนโดยตรง[/b] ครบทั้งเกม:

[list]
[*]เนื้อเรื่องทั้ง [b]17 บท[/b]
[*]เมนู · ระบบ · ไอเทม · สกิล · พรสวรรค์ · คำอธิบายตัวละคร · UI ทั้งหมด
[*]ใช้สำนวนแบบ[b]นิยายกำลังภายใน[/b] (ข้า / เจ้า / ท่าน) ตลอดทั้งเกม
[*]ชื่อเฉพาะทับศัพท์เป็นไทย (唐门 → สำนักถัง, 江湖 → ยุทธภพ)
[*]แยกสำนวน[b]ชาย/หญิง[/b]ตามตัวละครที่พูดจริง (ขอรับ / เจ้าค่ะ)
[/list]

[b]ครอบคลุม 71,523 จาก 72,883 ข้อความในเกม = 98.13%[/b]

[line]

[size=5][b]มีอะไรใหม่ใน v2.0[/b][/size]

v2.0 เปลี่ยนวิธีทำงานทั้งระบบ จากเดิมที่ "เห็นข้อความจีนแล้วเอาไทยไปทับ"
มาเป็น [b]ใส่คำแปลตรงตามรหัสประจำบรรทัดของเกม[/b] ผลคือแปลได้ในจุดที่วิธีเดิมทำไม่ได้เลย:

[list]
[*][b]ปี 三 十月中旬[/b] → [b]ปีที่ 3 กลางเดือนตุลาคม[/b]
[*]ปุ่มแนวตั้ง [b]修 / 练[/b] → [b]ฝึก / ฝน[/b]
[*]ตัวเลือก [b]食 · 衣 · 住 · 行[/b] → [b]อาหาร · เสื้อผ้า · ที่อยู่ · เดินทาง[/b]
[*][b]近战伤害10  爆击率0 %[/b] → [b]ความเสียหายประชิด 10  อัตราคริติคอล 0%[/b]
[/list]

พร้อมแก้สำนวนชาย/หญิงเพิ่มอีก 66 จุด และทำให้ม็อด[b]รอดจากการอัปเดตเกมในอนาคต[/b]

[line]

[size=5][b]วิธีติดตั้ง[/b][/size]

[b]1.[/b] โหลดไฟล์จากแท็บ [b]FILES[/b] แล้วแตกไฟล์ (extract)
จะได้ [b]BepInEx[/b], [b]winhttp.dll[/b], [b]doorstop_config.ini[/b] และไฟล์อื่น ๆ

[b]2.[/b] เปิดโฟลเดอร์เกม
[list]
[*]ปกติอยู่ที่ [b]C:\Program Files (x86)\Steam\steamapps\common\Legend_of_Mortal\[/b]
[*]หาไม่เจอ → คลิกขวาเกมใน Steam → [b]Properties → Installed Files → Browse[/b]
[/list]

[b]3.[/b] คัดลอก[b]ทุกอย่างที่แตกออกมา[/b] ไปวางในโฟลเดอร์เกม
วางถูกแล้วจะเห็น [b]winhttp.dll[/b] อยู่ระดับเดียวกับ [b]Mortal.exe[/b]

[b]4.[/b] เปิดเกม → หน้า Title เลือก[b]ออปชั่นที่ 2[/b]

[b]5.[/b] คลิกดรอปดาวน์ด้านบน → เลือก[b]ออปชั่นที่ 2 (จีนตัวย่อ / Simplified Chinese)[/b]
ระบบจะสวมทับด้วยภาษาไทยให้เอง

[quote][b]อัปเกรดจากเวอร์ชันเก่า:[/b] แนะนำให้[b]ลบโฟลเดอร์ BepInEx เดิมทิ้งก่อน[/b]
แล้วค่อยวางชุดใหม่ (ไฟล์ค้างเก่าอาจทำให้แปลเพี้ยน)
[b]เซฟเกมอยู่คนละที่ ไม่ได้รับผลกระทบ[/b][/quote]

[line]

[size=5][b]วิธีถอนการติดตั้ง[/b][/size]

ลบ [b]BepInEx[/b], [b]winhttp.dll[/b], [b]doorstop_config.ini[/b] ออกจากโฟลเดอร์เกม
[b]ม็อดนี้ไม่แก้ไฟล์ของตัวเกมเลยแม้แต่ไบต์เดียว[/b] ทุกอย่างเกิดขึ้นในหน่วยความจำตอนเล่นเท่านั้น

ถ้าอยากเก็บคำแปลไว้แต่ไม่เอาระบบใหม่ของ v2.0 ก็ลบแค่โฟลเดอร์
[b]BepInEx\plugins\LomThaiText[/b] โฟลเดอร์เดียว

[line]

[size=5][b]แก้คำแปลเองได้[/b][/size]

เจอคำที่ยังไม่สวย? แก้เองได้เลย แล้ว[b]กด Ctrl+F10 ในเกมเห็นผลทันที ไม่ต้องปิดเกม[/b]

[list]
[*][b]แก้คำแปล[/b] → [b]BepInEx\plugins\LomThaiText\Thai.tsv[/b]
แต่ละบรรทัดคือ [i]รหัส[/i] + [i]แท็บ[/i] + [i]คำแปลไทย[/i] — แก้เฉพาะฝั่งขวา บันทึกเป็น UTF-8
[*][b]แก้ตัวหนังสือเล็กไป / ล้นกล่อง[/b] → [b]BepInEx\Translation\th\Text\UI.resizer.txt[/b]
รูปแบบ [b]/เส้นทางของกล่อง=AutoResize(true,ขนาดต่ำสุด,ขนาดสูงสุด)[/b]
ไม่ต้องเดาเส้นทางเอง — ระหว่างเล่น ม็อดจะเขียนรายการกล่องที่ยังไม่ได้ตั้งค่าไว้ให้ที่
[b]BepInEx\LomThaiText_unruled.tsv[/b] พร้อมเส้นทางและขนาดจริง
[/list]

[size=5][b]English[/b][/size]

Unofficial [b]Thai[/b] translation for [b]Legend of Mortal (活侠传)[/b], translated
directly from Chinese — all 17 story chapters, menus, items, skills and UI.
[b]98.13% of the game's 72,883 text entries.[/b]

Install: extract and copy everything into the game folder (next to Mortal.exe),
launch, pick the 2nd title-screen option, then the 2nd dropdown option
(Simplified Chinese). BepInEx is bundled — no other requirements.
To uninstall, delete BepInEx, winhttp.dll and doorstop_config.ini.
[b]No game file is ever modified.[/b]
