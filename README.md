## 🧩 v2.0 — เปลี่ยนวิธีแปลทั้งระบบ

ตั้งแต่ v1.0 ถึง v1.6 ม็อดนี้ทำงานแบบเดียว: **เห็นข้อความจีนบนจอ แล้วเอาคำแปลไทยไปทับ**
(ผ่าน XUnity.AutoTranslator ซึ่งค้นด้วย "ข้อความ")

v2.0 เพิ่ม **ปลั๊กอิน `LomThaiText`** ที่เขียนคำแปลไทย **ลงในตารางข้อความของเกมโดยตรง โดยอ้างอิงรหัสประจำบรรทัด**
ข้อความทุกตัวในเกมถูกดึงผ่านรหัสอย่าง `Story/D_1_1_011` หรือ `System/Start/StartGame` อยู่แล้ว
ปลั๊กอินแค่เขียนทับตารางนั้นในจังหวะที่เกมสร้างเสร็จพอดี — **ไม่แก้ไฟล์เกมแม้แต่ไบต์เดียว**

> **คำแปลไม่ได้ถูกทำใหม่** — 71,412 จาก 71,523 บรรทัดคือคำแปลชุดเดิมจาก v1.6 ทั้งหมด
> เปลี่ยนแค่ "ท่อ" ที่ส่งคำแปลเข้าเกม แปลเพิ่มด้วยมือแค่ **111 บรรทัด** ที่วิธีเดิมเก็บไม่ได้จริง ๆ
> **ครอบคลุม 71,523 / 72,883 รหัส = 98.13%**

### สามอย่างที่วิธีเดิมทำไม่ได้ตลอดกาล

**1. ข้อความซ้ำที่ต้องแปลต่างกัน — 12,761 รหัส (17% ของทั้งเกม)**

คำว่า `好。` ในเกมมี **214 รหัสแยกกัน** พูดโดยคนละตัวละคร คนละสถานการณ์
แต่วิธีเดิมค้นด้วย *ข้อความ* ทั้ง 214 จุดจึงถูกบังคับให้แปลเหมือนกันชั่วนิรันดร์ — ตอนนี้แยกกันได้แล้ว

**2. ข้อความที่เป็นอักษรจีนตัวเดียว**

`唐` = "ถัง" แต่ถ้าใส่ไว้ ระบบเดิมจะไปแทนที่ในคำอื่นด้วย → `三师兄` กลายเป็น `สาม师兄`
เพราะแบบนี้ v1.2 จึงต้อง **ปิดการแปลอักษรเดี่ยวทิ้ง 97 บรรทัด** วิธีใหม่ไม่มีการค้นหาข้อความเลยจึงปลอดภัยสนิท

| ก่อน | หลัง |
|---|---|
| ปี 三 十月中旬 | **ปีที่ 3 กลางเดือนตุลาคม** |
| 修 / 练 (ปุ่มแนวตั้ง) | **ฝึก / ฝน** |
| 锻 / 造 | **ตี / เหล็ก** |
| 食 · 衣 · 住 · 行 (ตัวเลือก) | **อาหาร · เสื้อผ้า · ที่อยู่ · เดินทาง** |
| 唐门暗器总纲 (6 ช่องซ้อน) | **สำนัก·ถัง·อาวุธ·ลับ·ตำรา·ใหญ่** |

**3. ข้อความที่มีตัวเลขเปลี่ยนได้**

`近战伤害{0:N0}  爆击率{1:P0}` — ตัวเลขเปลี่ยนทุกครั้งจึงไม่มีทางจับคู่ได้
เอกสารเก่าเคยบันทึกไว้ว่า "รูปแบบไม่จำกัด แก้ไม่ได้" ตอนนี้ **บรรทัดเดียวครอบคลุมทุกค่า**
→ `ความเสียหายประชิด 10  อัตราคริติคอล 0%`

### 🛡️ รอดจากการอัปเดตเกม

**วิธีเดิมพังเงียบ ๆ** — ถ้าผู้พัฒนาแก้ประโยคจีนแม้แต่ตัวเดียว บรรทัดนั้นหยุดจับคู่ทันที
แล้วตกไปให้ Google แปลมั่ว โดยไม่มีอะไรเตือน

**วิธีใหม่** — รหัสประจำบรรทัดไม่เปลี่ยนตามเนื้อหา คำแปลเดิมยังทำงานต่อ และมีเครื่องมือ
เทียบว่าอะไรเปลี่ยนบ้าง แยกเป็น 3 กอง:

| กอง | ความหมาย |
|---|---|
| **ADDED** | รหัสใหม่ → แปลเพิ่ม |
| **CHANGED** | รหัสเดิมแต่**ต้นฉบับจีนถูกเขียนใหม่** → คำแปลไทยเก่าไปแล้ว ต้องแปลซ้ำ |
| **REMOVED** | รหัสถูกลบ |

กอง **CHANGED** คือสิ่งที่วิธีเดิมมองไม่เห็นเลย

### 🚻 สำนวนชาย/หญิงรายบรรทัด

v1.5 ดึงข้อมูล "ใครพูดบทไหน" จากสคริปต์บทของเกม (Lua) มาแล้ว แต่แก้ได้จำกัด
เพราะบรรทัดที่หลายตัวละครใช้ร่วมกันแยกไม่ออก — รอบนี้แยกได้ จึงแก้เพิ่มอีก **66 จุด**
(หญิงพูดแบบชาย 61 · ชายติดสำนวนหญิงมาจากรอบก่อน 5)

พร้อมกันนี้พบว่า **ข้อมูลเพศเดิมตกหล่นตัวละครหญิงไป 29 คน** — รวม **叶云裳 (เย่หวินฉาง),
温夫人 (ท่านผู้หญิงเวิน), แม่ของ 赵活 และกลุ่มศิษย์หญิงสำนักถัง** ตอนนี้บันทึกครบพร้อมหลักฐานรายคน

> 🆕 v2.0 เพิ่มโฟลเดอร์ `BepInEx\plugins\LomThaiText\` มาด้วย (ปลั๊กอิน + ตารางคำแปล)
> **ไม่ชอบก็ลบโฟลเดอร์นี้ทิ้งโฟลเดอร์เดียว** จะกลับไปเป็นพฤติกรรมแบบ v1.6 ทันที
> ปลั๊กอิน**ไม่แก้ไฟล์ของตัวเกมเลย** ทุกอย่างเกิดในหน่วยความจำตอนเล่นเท่านั้น
> ⚠️ ตัวเกมเป็น **32-bit** และค่า `FromLanguage` ต้องเป็น `zh-CN` (ตั้งมาให้แล้ว)

## 🔤 ฟอนต์ / Font

ตั้งค่าใช้ **`OverrideFont=Tahoma`** มาให้แล้วในไฟล์ `BepInEx/config/AutoTranslatorConfig.ini` (แสดงผลภาษาไทยรวมวรรณยุกต์ซ้อนได้ดี ไม่ต้องลงฟอนต์เพิ่ม)

## 🐞 พบข้อผิดพลาด?

- **ตัวหนังสือล้นกล่อง / เล็กเกินไป** → แก้ที่ `BepInEx/Translation/th/Text/UI.resizer.txt`
  รูปแบบ `/เส้นทางของกล่อง=AutoResize(true,ต่ำสุด,สูงสุด)` แล้วกด **Ctrl+F10** ในเกมเพื่อดูผลทันที
  **ไม่ต้องเดาเส้นทางเอง** — ระหว่างเล่น ปลั๊กอินจะเขียนรายการกล่องที่ยังไม่ได้ตั้งค่าไว้ให้ที่
  `BepInEx/LomThaiText_unruled.tsv` พร้อมเส้นทางและขนาดกล่องจริง
- **เจอคำแปลที่ยังไม่สวย** → แก้ใน `BepInEx/plugins/LomThaiText/Thai.tsv` (บันทึกเป็น UTF-8,
  ห้ามแก้ฝั่งซ้ายซึ่งเป็นรหัส) แล้วกด **Ctrl+F10** เห็นผลเลยโดยไม่ต้องปิดเกม
- **ตัวอักษรเป็น □ (tofu)** → ตรวจว่า `OverrideFont` ตั้งเป็นฟอนต์ที่มีสระ/วรรณยุกต์ไทย
- ปิดตัวเลือก *"Enable color change of read text"* ในเกม ไม่งั้นบางข้อความอาจไม่ถูกแทนที่
