🔧 วิธีติดตั้งแบบย่อ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. โหลดไฟล์ zip แล้วแตกไฟล์
2. เปิดโฟลเดอร์เกม → คลิกขวาเกมใน Steam → Properties → Installed Files → Browse
3. คัดลอกทุกอย่างที่แตกออกมา ไปวางในโฟลเดอร์เกม
   (วางถูกแล้วจะเห็น winhttp.dll อยู่ระดับเดียวกับ Mortal.exe)
4. เปิดเกม → หน้า Title เลือก "ออปชั่นที่ 2"
5. คลิกดรอปดาวน์ด้านบน → เลือก "ออปชั่นที่ 2" (จีนตัวย่อ / Simplified Chinese)
   ระบบจะสวมทับด้วยภาษาไทยให้เอง

ไม่ต้องลงโปรแกรมอะไรเพิ่ม ตัว mod loader แถมมาในไฟล์แล้ว

❌ ถอนการติดตั้ง: ลบ BepInEx, winhttp.dll, doorstop_config.ini ออกจากโฟลเดอร์เกม
   ม็อดนี้ไม่แก้ไฟล์ของตัวเกมแม้แต่ไบต์เดียว เซฟเกมไม่ได้รับผลกระทบ

• ปุ่มแนวตั้ง 修 / 练 → ฝึก / ฝน
• ตัวเลือก 食 衣 住 行 → อาหาร เสื้อผ้า ที่อยู่ เดินทาง
