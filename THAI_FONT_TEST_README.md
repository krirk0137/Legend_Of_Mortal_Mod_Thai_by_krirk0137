# 🧪 ทดสอบฟอนต์ไทย — Legend of Mortal (รอบนำร่อง)

จุดประสงค์ของรอบนี้ **ไม่ใช่การแปลจริง** แต่คือทดสอบว่า **เกมแสดงตัวอักษรไทยได้หรือไม่**
ผมแปลเมนูบางส่วนเป็นไทยไว้ให้เห็นชัดๆ ตอนเปิดเกม

---

## ✅ ผมแก้อะไรไปบ้าง

1. **แปลเมนูหลัก ~36 คำเป็นไทย** (ในไฟล์ `Translation zh-CN to en.txt`)
   ตัวอย่าง: `启程→ออกเดินทาง`, `环境设定→ตั้งค่า`, `继续游戏→เล่นต่อ`,
   `确定→ยืนยัน`, `取消→ยกเลิก`, `属性→คุณสมบัติ`, `技能→ทักษะ`,
   `行囊→ย่ามสัมภาระ`, `门派→สำนัก`, `状态→สถานะ`, `系统→ระบบ`, `战斗→การต่อสู้` ฯลฯ
2. **ตั้งฟอนต์** `OverrideFont=Leelawadee UI` (ฟอนต์ไทยของ Windows) — ใช้ได้กับ UI แบบ UGUI
3. **เปิดหน้าต่าง Console** เพื่อดู log ระหว่างทดสอบ

> 💾 ไฟล์อังกฤษต้นฉบับถูกสำรองไว้แล้วที่
> `Translation zh-CN to en.txt.EN-BACKUP` — กู้คืนได้ตลอด

---

## ▶️ วิธีทดสอบ

1. **ก๊อปปี้ไฟล์ที่แก้ไป 2 ไฟล์นี้ ทับลงในโฟลเดอร์เกมจริง** (ที่ติดตั้งผ่าน Steam)
   - `Mod\BepInEx\Translation\en\Text\Translation zh-CN to en.txt`
   - `Mod\BepInEx\config\AutoTranslatorConfig.ini`

   > ถ้าง่ายกว่า ก๊อปทั้งโฟลเดอร์ `Mod\` ทับลงในโฟลเดอร์เกมได้เลย

2. **เปิดเกม** → เข้าหน้า Title Screen (เลือกตัวเลือกที่ 2 / เลือกภาษาจีนตัวย่อ ตามปกติ)
3. **ดูที่ปุ่มเมนู** เช่น "ออกเดินทาง / ตั้งค่า / เล่นต่อ" และแท็บเมนูในเกม (คุณสมบัติ / ทักษะ / สถานะ ฯลฯ)

---

## 🔎 อ่านผล — มี 3 แบบ

| สิ่งที่เห็น | แปลว่า | ทำต่อยังไง |
|---|---|---|
| 🟢 **ตัวไทยขึ้นสวย อ่านออก** | ฟอนต์ผ่าน! ใช้ทางนี้ได้ | เดินหน้าแปลจริง (อังกฤษหลัก + แทรกไทย) |
| 🟥 **เป็นสี่เหลี่ยม □□□ / ว่าง** | ฟอนต์ไม่มีตัวไทย (น่าจะเป็น TextMeshPro/FairyGUI) | ต้องทำ **Thai font asset bundle** — บอกผม เดี๋ยวจัดการต่อ |
| ⬜ **ยังเป็นภาษาจีนเหมือนเดิม** | ตัวหนังสือไม่ถูก hook / คีย์ไม่ตรง | ส่ง log มาให้ดู |

---

## 📮 สิ่งที่ขอให้ส่งกลับมา

1. **ภาพหน้าจอ** ของหน้าเมนู (title screen + แท็บเมนูในเกม)
2. ไฟล์ **log**: `Mod\BepInEx\LogOutput.log`
3. ในหน้าต่าง Console สีดำ ถ้ามีบรรทัดสีแดง/คำว่า `font` `Fallback` `Error` ถ่ายรูปมาด้วย

---

## ↩️ วิธีย้อนกลับเป็นอังกฤษล้วน (ถ้าต้องการ)

1. ลบไฟล์ `Translation zh-CN to en.txt` แล้วเปลี่ยนชื่อ `...EN-BACKUP` กลับเป็นชื่อเดิม
2. ในไฟล์ config: ตั้ง `OverrideFont=` (ว่าง) และ `EnableConsole=False`
